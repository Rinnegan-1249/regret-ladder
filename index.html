01 /What this site is

This project builds a ladder of increasingly capable game-solving algorithms — starting from regret matching on a normal-form game and climbing to the CFR family on an imperfect-information poker game — with every step implemented from scratch, validated against OpenSpiel's reference solvers, and accompanied by reproducible experiments. This site lets you play against the actual trained bots, watch the algorithms train live, and read the theory behind each rung, following the primary literature.

Rung 1Normal-form games, Nash equilibria, best responses
Rung 2Regret matching on Rock–Paper–Scissors (Hart & Mas-Colell 2000)
Rung 3Vanilla CFR on Kuhn Poker (Zinkevich et al. 2007)
Rung 4CFR+ (Tammelin 2014) and Monte Carlo CFR (Lanctot et al. 2009)
Rung 5Scaling the full CFR family to Leduc Poker
NextAbstraction, simplified Hold'em
This is the static version of the site — fast to load, but play is a recorded walkthrough rather than a live match, and training graphs replay a precomputed run. For full interactive play and real-time training, try the dynamic version (it spins up a live Python server on first visit, so the first load is slower).
